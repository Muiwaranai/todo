Welcome to the Todo API

Here are the available endpoints:

  • GET /api/getall/ - Get all tasks
  • GET /api/get?id=N - Get task by ID, N should be Id from 1 to Integer Max value
  • POST /api/addtask - Add a new task, Takes json data this this format -
  • REMOVE /api/remove?id=N - Remove task by ID, N should be Id from 1 to Integer Max value
